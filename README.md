# U1-tipo-ENUMerado
tipo de datos enum, para selector



```java
     public class TipoENUM {
         public static void main(String[] args) {
            enum Clase { ALFA, BRAVO!, CHARLIE }; 
            Clase valor; 
            valor= Clase.BRAVO!; 
            System.out.println("el valor es:  "+valor);
         }
     }
```
